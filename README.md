# 常用开发模块集合
本项目是一个模块化的开发集合，包含了常用的业务模块，旨在提高开发效率，减少重复开发工作。每个模块都是独立且可复用的，可以根据实际需求快速集成到项目中。
##  技术栈
### 前端技术栈
- **Node.js**: v20.19.5
- **Vue**: 3.x
- **构建工具**: Vite 7.x
- **其他**: QRCode（二维码生成）
### 后端技术栈
- **Java**: 21
- **Spring Boot**: 3.2.5
- **数据库**: MySQL 8.x
- **ORM框架**: MyBatis Plus 3.5.5
- **安全框架**: Spring Security + JWT
- **缓存**: Redis（Lettuce连接池）
- **邮件服务**: Spring Mail（支持QQ邮箱）
- **虚拟线程**: Java 21 Virtual Threads
## 模块列表
### LoginRegister_Mod - 登录注册模块一个功能完整的用户认证模块，支持多种登录方式和第三方OAuth登录。
